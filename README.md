# 🍔 Zomato Food Delivery Analytics Dashboard | Power BI

## 📊 Project Overview

This project presents an interactive **Zomato Food Delivery Analytics Dashboard** built using **Power BI** to analyze food ordering patterns, customer behavior, payment preferences, and sales performance.

The dashboard transforms raw food delivery data into meaningful business insights, helping stakeholders make data-driven decisions and improve operational efficiency.

---

## 🚀 Key Features

### 📈 Sales Performance Analysis

* Month-wise sales trend analysis
* Quantity sold tracking across different periods
* Identification of peak and low-performing months

### 🍽️ Food Item Analysis

* Top-selling food items visualization
* Quantity distribution by food category
* Product performance comparison

### 👥 Customer Insights

* Gold vs Regular member analysis
* Order quantity segmentation
* Customer category performance tracking

### 💳 Payment Method Analysis

* UPI, Card, and Cash on Delivery (COD) comparison
* Payment preference trends
* Revenue contribution by payment type

### 🚚 Delivery Performance Monitoring

* Delivered vs Cancelled order analysis
* Delivery success rate visualization
* Operational efficiency tracking

### 🎛️ Interactive Filters

* Month Filter
* Member Type Filter
* Payment Method Filter
* Restaurant Type Filter

## 📊 Dashboard Insights

### Key Findings

* Samosa emerged as the highest-selling food item.
* UPI is the most preferred payment method among customers.
* February recorded the highest sales volume.
* Delivery success rate exceeded 94%.
* Gold Members contributed significantly to total orders.
